Library Sections


Acquisition Section

The College Library has an Acquisition Section where new documents are procured and then processed as per standards and norms of the library. The main function includes preparation of book list, duplicate checking, ordering, accessioning of books and processing bills for final payment. All the books received from vendors/publishers are physically processed and accessioned in this section.

Technical Section

Technical Section which is located on the first floor is responsible for classification and cataloguing of the accessioned documents. In this section every document is entered into KOHA database before barcoding. Dewey Decimal Classification system (23rd edition) is adopted for the classification of all the documents and the cataloging is based on MARC format that follows Classified Catalogue Code

Circulation Desk

The College library has a separate counter for Circulation Desk which is computerized in all aspects. The circulation section is situated in the ground floor and it is automated through KOHA Library Management Software. Borrowing of documents is allowed against valid library cards as per library circulation rules.

The following duties are carried out by this section:
Membership Registration

Books issue and return using barcode scanners.

Reservation and Renewal of documents

Issue No Dues Certificate

Overdue Fine Collection

Alert Services such as overdue, collect notice etc
